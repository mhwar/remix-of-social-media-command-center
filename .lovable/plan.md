
# منصة إدارة طلبات السوشل ميديا

## نظرة عامة
منصة متكاملة لإدارة طلبات عملاء السوشل ميديا، تشمل إدارة المهام، تقويم المحتوى، ومتابعة الفريق والعملاء. واجهة عربية (RTL) بتصميم عصري ونظيف.

## الصفحات والمكونات

### 1. لوحة التحكم الرئيسية (Dashboard)
- إحصائيات سريعة: عدد العملاء، المهام النشطة، المهام المكتملة، المهام المتأخرة
- قائمة آخر المهام والطلبات
- نظرة على الأنشطة الأخيرة

### 2. إدارة العملاء (Clients)
- قائمة العملاء مع بحث وفلترة
- ملف كل عميل: الاسم، الشركة، منصات السوشل ميديا، معلومات التواصل
- عرض طلبات ومهام كل عميل

### 3. إدارة الطلبات والمهام (Orders/Tasks)
- إنشاء طلب جديد بنوع الخدمة: كتابة محتوى، تصميم، تقرير أداء، إدارة إعلانات
- تعيين المهمة لعضو الفريق مع تاريخ التسليم
- حالات المهام: جديد → قيد التنفيذ → بانتظار موافقة العميل → مكتمل
- إرفاق ملفات (التصاميم/المحتوى)
- تعليقات وملاحظات على كل مهمة

### 4. تقويم المحتوى (Content Calendar)
- عرض شهري للمحتوى المجدول لكل عميل
- سحب وإفلات لتعديل المواعيد
- تمييز بالألوان حسب نوع المحتوى والحالة

### 5. إدارة الفريق (Team)
- عرض أعضاء الفريق (1-3 أشخاص)
- المهام الموكلة لكل عضو وحالتها
- أدوار: مدير، كاتب محتوى، مصمم

### 6. بوابة العميل (Client Portal)
- واجهة مبسطة يدخلها العميل لمتابعة طلباته
- عرض المحتوى بانتظار الموافقة مع أزرار قبول/رفض مع ملاحظات

## التصميم
- واجهة عربية كاملة (RTL) بتصميم حديث ونظيف
- ألوان احترافية مع sidebar للتنقل
- متجاوبة مع الموبايل
- أيقونات واضحة وتنقل سلس

## التقنيات
- React + Tailwind CSS + shadcn/ui
- Lovable Cloud للقاعدة والمصادقة
- تسجيل دخول بالإيميل وكلمة المرور
- صلاحيات: مدير، عضو فريق، عميل
